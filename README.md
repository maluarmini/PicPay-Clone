# Tecnologias

- Styled Components

# Como usar
-  Clone o repositório: git clone https://github.com/maluarmini/PicPay-Clone.git
-  Entro no repositorio: cd PicPay-Clone
-  Instale as dependencias: yarn install
-  Inicie a aplicação: yarn start
